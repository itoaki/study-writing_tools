# writing_tools
執筆用のツールをおいていきます。

## リポジトリのチェックアウト

~~~
git clone git@github.com:itoaki/writing_tools.git
~~~

## テンプレートの生成

~~~
cd writing_tools
sh review-init.sh
~~~

プロジェクトの名前を聞かれますので、任意の名前を入力します。

## PDFファイルの生成

~~~
cd writing_tools
sh review-pdfmaker.sh
~~~

プロジェクトの名前を聞かれますので、PDFを生成したいプロジェクトの名前を入力します。
プロジェクトファイルの中に、book.pdfというファイルが生成されます。
